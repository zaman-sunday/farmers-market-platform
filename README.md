# Farmers Market Platform

A comprehensive smart contract platform for managing local farmers markets built on Stacks blockchain.

## Features

- **Vendor Registration**: Farmers and vendors can register and get verified on the platform
- **Space Management**: Market organizers can create and allocate booth spaces with custom pricing
- **Payment Processing**: Secure STX-based rental payments with automatic transfers
- **Customer Reviews**: Rating and review system for vendor feedback and quality assurance
